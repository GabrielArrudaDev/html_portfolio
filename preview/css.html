Tags :

ID:

Pode ser usada apenas uma vez e na hora de utilizá-la na página do css temos que começar usando ''#'' para esclarecer que é um id e depois o nome da class que colocamos.

CLASS:

Pode ser usada em todas as páginas e na hora de utilizá-la na página do css temos que começar usando ''.'' para esclarecer que é uma class e depois o nome da class que colocamos.

Como utilizar o ID e o CLASS no HTML:

ID:

<p id = "id_teste"> Lorem ipsum, dolor sit amet consectetur adipisicing elit. Debitis dolore dolor sapiente quidem rem praesentium quos eum repellat perspiciatis voluptates earum, et itaque quod placeat pariatur molestias qui sint veritatis!

CLASS:

<p class = "class_teste"> Lorem ipsum, dolor sit amet consectetur adipisicing elit. Debitis dolore dolor sapiente quidem rem praesentium quos eum repellat perspiciatis voluptates earum, et itaque quod placeat pariatur molestias qui sint veritatis!

Exemplos :

p{
color: red;
}

#id_teste{
color: blue;
}

.class_teste{
color: green;
}

Como deixar uma imagem de background responsiva:

Para deixar uma imagem de fundo responsiva, deve-se usar varios comandos e ir testando ambos, pois cada imagem tem algo mais importante para se mostrar. Exemplo: Imagem com um céu e uma ilha. Nesse caso o céu nao é tao importante quanto a ilha, sendo assim essa imagem tem como mais importante a ilha, sendo assim, caso seja um site responsivo, é necessario ir testando varios comandos para ver qual se encaixa melhor, ter varias imagens para tamanhos diferentes ou simplesmente fazer um corte na imagem por comandos para manter responsivo.

Comandos :

  • background-image: url(#);
    • esse comando é para colocar a imagem (nao pode ter espaço na parte do url)
  • background-attachment: fixed;
    • esse comando é para manter a imagem fixa, porem acaba cortando a parte de baixo.
  • background-size: #;
    • esse comando é para escolher o tamanho da imagem. caso escolha 100% a imagem vai pegar em tudo, porem quando diminuir o site, a imagem acaba ficando distorcida, fica cortando pra cima ou pra baixo, etc.
    • caso escolha content a imagem sempre ficara ali, porem ela vai ter que se repetir.
    • caso escolha o cover a imagem ficara inteira mesmo que esconda um pedaço.
  • background-repeat: #;
    • esse comando é para dizer se a imagem repete ou nao, e em qual lado.
  • background-position: #;
    • esse comando é para alinhar a imagem. Exemplo: right (horizontal) bottom (vertical). Deixando assim a imagem sempre aparecendo por inteira no fundo (ilha e o ceu) porem com uma parte do ceu cortada

Como ficou :

Body{
background-image: url(#);
background-attachment: fixed;
background-size: cover;
background-repeat: repeat;
background-position: right bottom;
}
